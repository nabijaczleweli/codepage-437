pub trait BorrowFromPcCp437<'c, T: ?Sized> {
    fn borrow_from_pc_cp437(cp437: &'c T) -> Self;
}

Try to borrow data encoded in cp437 as a Unicode container of the specified type.

If that cannot be done, clone it.
